Remarks

When using file, FilePath is updated and the Save( ) overload is called for processing.

Frame.FrameHeader.Shift value (see ACT_Image_File.txt for format reference) will either be 03, 04 pr 05, depending on number of colors and image width. The Shift OpCode (0xFB) will not be rewritten to improve compression.
