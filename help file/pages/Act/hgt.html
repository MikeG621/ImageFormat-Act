Act.Height

Namespace: Idmr.ImageFormat
Assembly: Idmr.ImageFormat.Act.dll
Syntax: public int Height { get; }

Gets the overall height of the Act object

Remarks

Individual frames may be different sizes, but Height is the total height when all frames are laid on top of each other relative to Center
