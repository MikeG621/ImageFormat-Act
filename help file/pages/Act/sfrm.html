Remarks

Image is converted to 8bppIndexed. Frame palette is trimmed to only the colors used in the image and the frame's NumberOfColors is updated. If SetFrame( ) has been called from the Act(Bitmap) constructor and the image is not Indexed, palette is the 256-color default.
